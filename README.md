# README

Initial and final coordinates for simulations described in "Structural and dynamic studies uncover a distinct allosteric modulatory site at the µ-opioid receptor" by Zhang et. al.,DOI: 10.1101/2025.10.14.682454

The coordinates are stored in a compressed (tarball) `.gro` format - a human-readable format used by the GROMACS,
compatible with most molecular visualization software (PyMOL, UCSF Chimera, VMD, etc.)

The initial coordinates for simulations of the BMS-986187-bound and the BMS-free complexes are
[bms/bms187_md_init.tgz](bms/bms187_md_init.tgz)
and [nobms/nobms_md_init.tgz](nobms/nobms_md_init.tgz) respectively. These coordinates were used to seed 20 replicates with randomized initial
velocities, and the final coordinates for each replicate are also provided.
